# Security Policy

Crash Detector reads local Windows Event Viewer data and selected executable paths. It should not upload crash logs, hardware data, usernames, file paths, or reports anywhere.

## Reporting Issues

Please report security concerns privately through the project maintainers or the Lumin community at https://discord.gg/lumin.

Do not open a public issue containing sensitive crash logs, full local file paths, usernames, dump files, access tokens, or other private machine data.

## Handling Local Data

When sharing reports, remove:

- Windows usernames.
- Full local paths that identify a person or organization.
- Machine names.
- Dump files or logs containing tokens.
- Proprietary application names unless disclosure is intended.
